Software para electrónicos: Kicad (2ª Parte)

En una entrada anterior de mi blog hablé acerca de kicad, un software libre para desarrollo electrónico. Prometí contar los resultados y bueno, lo prometido es deunda.... Pues eso, he estado utilizando kicad para el desarrollo de un proyecto real y he aquí mis conclusiones a la utilización de éste software:
  • Software muy intuitivo y fácil de utilizar.
  • Gran número de librerías de componentes y huellas además de la existencia de librerías portadas de Eagle.
  • La creación de librerías de componentes, de componentes nuevos así como de huellas es fácil y rápida.
  • La edición de propiedades así como la asignación de huellas a los distintos componentes se hace de forma mucho mas rápida a como se hacía, por ejemplo, con PCB.
  • El proceso de rutado es auto-optimizado obteniendo muy buenos resultados practicamente desde el primer momento, siendo necesarias muy pocas modificaciones posteriores.
  • Permite ver tridimensionalmente como quedará el PCB una vez montado (aproximadamente).
