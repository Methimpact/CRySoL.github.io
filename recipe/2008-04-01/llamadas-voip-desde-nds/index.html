Llamadas VoIP desde NDS

Si ya podemos conectar nuestra NDS a “la Fonera” ¿por qué no utilizar esa conexión para hacer algo más? Podemos hacer llamadas por VoIP a números fijos, sin ningún coste, y lo mejor de todo, DESDE NUESTRA NDS! gracias a SVSIP.

Instalación y configuración

Antes de descargarnos la aplicación, lo primero que necesitamos en una cuenta con alguno de los proveedores de telefonía IP, aquí encontraréis algunos junto con su configuración. (Nota: Skype no es válido porque no utiliza SIP).

Yo he utilizado VoipBuster, aunque aviso que para poder crear una cuenta de usuario con este proveedor hay que descargarse un .exe desde su página. La instalación no ha dado problemas con el wine, pero no he conseguido ejecutarlo…

Una vez que ya tenemos una cuenta con algún proveedor, pasamos a descarganos la aplicación SVSIP de la <a href= “http://svsip.free.fr/spip.php?rubrique17”> página del autor .

Una vez que hemos descargado y descomprimido el archivo, en el directorio svsip está el archivo config.txt, que tendremos que completar con los datos del proveedor de telefonía IP y de la cuenta de usuario, quedando de la siguiente manera:



    
	
  1. This is a comment in the config file.
    2. 
	
  2. Information on http://svsip.free.fr
    3. 



    
	
  1. Set domain sip provider (example: freephonie.net)
    
—domain sip.VoipBuster.com
    2. 



    
	
  1. Set authentication username
    
—username XXX
    2. 



    
	
  1. Set authentication password
    
—password XXX
    2. 



    
	
  1. Optional registration interval (default 55) (for freephonie set 1800)
    
—reg-timeout 1800
    2. 



    
	
  1. Optional microphone volume (default 1.0)
    
—mic-level 3.0
    2. 



    
	
  1. Nuevo control de altavoz (default 1.0)
    
—speaker-level 2.0
    2.

Finalmente, copiamos en el raíz de nuestra flashcard o tarjeta de memoria el directorio "svsip" y el ejecutable para la NDS "svsip.nds". Ahora, si no tenéis configurada la conexión al punto de acceso, el ejecutable "wifi-config.nds" que viene con el archivo comprimido descargado os ayudará a elegir y a configurar la conexión.

Y con esto, ya podéis hacer pruebas!
