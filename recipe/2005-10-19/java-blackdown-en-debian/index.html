Java Blackdown en Debian

Cómo instalar la máquina virtual Java de Blackdown en Debian GNU/Linux

AVISO: La JVM de Blackdown NO ES software libre.

Ingredientes

Blackdownwww.blackdown.org/ dispone de paquetes Debian. Para instalarlos incluir esta línea en el /etc/apt/sources.list

deb ftp://ftp.tux.org/java/debian/ sarge non-free

E instalar de la forma habitual:

# apt-get update
# apt-get install j2re1.4

También se puede instalar el SDK, paquete j2sdk1.4

Si tienes cualquier problema usando apt-get, puedes descargar los paquetes de <a href="ftp://ftp.tux.org/java/debian/pool/non-free/j/j2se1.4-i586/">ftp://ftp.tux.org/java/debian/pool/non-free/j/j2se1.4-i586/@

Java plugin para Mozilla/Firefox/etc

Si no existe (que puede que si) hay que hacer un enlace desde el directorio de plugins del navegador a la librería del plugin. Para Firefox se haría:

# ln -s /usr/lib/j2re1.4-blackdown/jre/plugin/i386/ns7/libjavaplugin_oji.so \
          /usr/lib/mozilla-firefox/plugins/
