SELECT

SELECT <fields> FROM <tables> [WHERE <condition>];
Ejemplo: 
SELECT * FROM tabla_elementos WHERE id_elementos = 3;

INSERT

INSERT INTO <table> (<list of columns>) VALUES (<list of values>);
La lista de columnas y la de valores debe ir en el mismo orden. Ejemplo: 
INSERT INTO tabla_elementos (id_elementos, nombre) VALUES (1, "elem1") ;

UPDATE

UPDATE <table> SET <list of assignations> [WHERE <condition>];
Ejemplo: 
UPDATE tabla_elementos SET nombre="elem_1", valor="3" WHERE id_elementos = 1;

DELETE

DELETE FROM <table> [WHERE <condition>];
Ejemplo: 
DELETE FROM tabla_elementos WHERE id_elementos = 1;
