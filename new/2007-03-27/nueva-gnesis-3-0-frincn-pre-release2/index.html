Nueva GNESIS 3.0 "fRincón" Pre-Release2

En la página web de GNESIS (en la sección descargas) podeis encontrar la nueva Pre-Release que se ha generado con las siguientes novedades:

  • Entornos gráficos XFCE y GNOME, a elegir.
  • Mucha mayor personalización de la Debian-Live original.
  • OpenOffice y Iceweasel (castellanizados)
  • Utilidades para conectividad wireless.
  • Entorno completo de programación en C.

Existe un error conocido que vereis al arrancar el asistente para la configuración de un usuario personal. El asistente es legible, pero las ventanas de dialog no se dibujan del todo bien. Este error está en proceso de reparación. Además, como es persistente sólo lo tendreis que ejecutar una vez :-).

Recomiendo que probeis ambas y que escojais la que más os guste. Personalmente, la versión en XFCE es rapidísima y me gusta mucho más. Pero para gusto, colores.

Espero vuestras propuestas, sugerencias y errores que podais encontrar. Gracias por vuestra atención.

IMPORTANTE: consultad como instalar GNESIS en un dispositivo USB si no lo habeis mirado aún.

¡Un saludo!
