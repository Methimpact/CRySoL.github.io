CRySoL GNU Install Party v3.2

Tengo el gusto de informaros de la celebración de la próxima Install Party de CRySoL que se celebrará en la Escuela Superior de Informática los días 22 y 23 (viernes y sábado).

La Install Party se orientará principalmente a los alumnos que cursen la asignatura Estructura de Computadores, teniendo preferencia para la asignación de puestos, aunque la party está abierta a todo el mundo.

Para confirmar asistencia (y reservar puesto) debéis mandar un correo a: mariajose.santofimia@uclm.es. Podéis indicar en el mail características especiales del hardware que vayáis a llevar.

Los requisitos mínimos serán 10GB de disco SIN PARTICIONAR (o una partición de 10GB que será la que usemos).

A lo largo de la semana iremos confirmando más talleres, de momento serán los siguientes:

  • Primera sesión (Viernes, 18:00h):
    • Instalación del sistema base. Primeros pasos.
    • Herramientas de desarrollo para NDS.
  • Segunda sesión (Sábado, 10:30h):
    • Herramientas de desarrollo para PSP.
    • Redes P2P. Torrentflux.
